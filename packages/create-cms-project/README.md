# create-cms-project

```shell
Usage: create-cms-project <cms-project-name> [options]

Options:
  -v, --version       output the version number
  -r, --repo [repo]   specify CMS boilerplate git repo to clone (default: "HubSpot/cms-theme-boilerplate")
  -s, --ssh           use SSH to clone CMS boilerplate git repo
  -i, --skip-install  skip installation of boilerplate deps
  -h, --help          output usage information
```
